## 🚀 Hoe Ga Je Nu Verder?

### Stap 1: Domein Registreren (NU)
```
Koop: montrelle.fr
Via: OVHcloud.com, Gandi.net, of Namecheap
```

### Stap 2: Shopify Account
1. Ga naar shopify.com
2. Start free trial
3. Kies plan: Basic ($39/maand)

### Stap 3: Theme Upload
```bash
cd dropship-store/dropship-theme
zip -r montrelle-theme.zip .
# Upload naar Shopify Admin → Themes
```

### Stap 4: Producten Importeren
1. Shopify Admin → Products
2. Import → Upload `products.csv`
3. Check resultaten

### Stap 5: Collecties Maken
| Collectie | Condition |
|-----------|-----------|
| Femme | Product tag = femme |
| Homme | Product tag = homme |
| Nouveautés | Created date |
| Promotions | Compare at price > 0 |

### Stap 6: Thema Configureren
- Logo uploaden (wanneer je die hebt)
- Homepage hero afbeelding
- Social media links
- Menu's aanmaken
- Kleuren & fonts checken

### Stap 7: Panda Koppeling
1. DSers app installeren
2. Panda toevoegen als leverancier
3. Producten linken aan Panda catalogus

### Stap 8: Launch!
- Betaalmethodes testen
- Test bestelling plaatsen
- Legal pages publiceren
- Social media handles claimen
